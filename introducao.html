Introdução Lembrando que esse é um tutorial voltado para iniciantes em desenvolvimento de jogos, logo, vou abordar aqui alguns princípios básicos desse universo em relação à parte de engenharia (programação), apresentando uma ferramenta, não tão famosa, chamada Pyxel. O Pyxel é uma game engine (motor de jogos) para criação de jogos com visual mais retrô, distribuído como uma biblioteca do Python. Não sabe o que é uma biblioteca nem o que é Python? Então tenha calma! Eu vou te explicar, mas considere buscar ampliar seus conhecimentos sobre esse vasto universo! O Python é uma linguagem de programação, ou seja, um dialeto para se comunicar com o computador e dar-lhe instruções de como executar determinada tarefa. Em outras palavras, é uma língua própria para programar um computador a fim de que ele realize algo. As bibliotecas são conjuntos de várias instruções já criadas por algum programador que podem ser utilizadas por você. O Pyxel é uma dessas bibliotecas. Ele já vem com diversas funcionalidades prontas que podem ser usadas assim que instalado, como: Desenhar formas geométricas; Criar sons e músicas; Projetar tilemaps (conjunto de sprites usado para compor cenários); Ler entradas de teclado, mouse e gamepad; Etc. Viu só? Ele é uma ferramenta bastante interessante para criar jogos quando se está iniciando, possuindo um nível de entendimento bem simples, até para quem é iniciante.

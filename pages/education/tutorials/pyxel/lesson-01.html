Configurando ambiente A primeira coisa a fazer antes de iniciar a criação com o Pyxel é configurar o ambiente de desenvolvimento, ou seja, instalação e preparação das ferramentas que vamos utilizar para programar o nosso jogo. Todas elas são gratuitas e podem ser encontradas nos links abaixo. As ferramentas que vamos baixar e utilizar neste tutorial são: um editor de código (no meu caso, o Notepad++), o Python (como linguagem de programação) e o Pyxel (que, como mencionado na aula anterior, é uma biblioteca do Python). Algumas outras ferramentas podem ser vistas mais para frente, mas o essencial para nosso desenvolvimento é isso! Para instalar o Python, você pode acessar o link e facilmente realizar o download da linguagem na sua máquina. Em seguida, devemos testar se a instalação foi bem sucedida. Abra o prompt de comando, digitando Win + R e na caixa de texto, digite "cmd". Ou abra diretamente pelo menu iniciar. Para testar a instalação digite o seguinte comando no prompt: python --version Quando instalado o Python, vá até o prompt de comando do seu SO e tente digitar: pip install pyxel Esse comando é o responsável por fazer a instalação do Pyxel. O pip é o instalador de bibliotecas para Python. Com apenas esse comando você diz a ele para trazer todas as funcionalidades incluídas no Pyxel. Atenção! Erros são muito comuns durante a instalação de ferramentas de desenvolvedor. O importante é não se desesperar e não desistir. Caso possível, tente ler e entender do que se trata o erro para resolvê-lo. Também pode tentar copiar e colar a mensagem de erro no seu navegador. Muitos sites e fóruns hoje estão disponíveis para suporte. Além do link para download, estou deixando também os links para as documentações dessas ferramentas, que podem ser usadas como referência.

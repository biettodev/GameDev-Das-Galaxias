Pyxel: O Pyxel é uma game engine (motor de jogos) retrô, distribuída como um pacote do Python. Ele permite a criação de jogos baseados em paletas de apenas 16 cores, garantindo a fidelidade com as especificações dos consoles de jogos antigos, além de outras funcionalidades muito interessantes, como:

  • Criação de fontes de recursos, tais como imagens;
  • Editor de pixels;
  • Editor de tilemap;
  • Editor de som e de música.

O repositório do projeto pode ser acessado em: https://github.com/kitao/pyxel

Considero muito interessante esse tipo de ferramenta para aprender, principalmente se seu foco é construir sua própria engine.

O repositório do Pyxel possui tradução para Português e tem suporte para Windows, Mac, Linux Ubunto e alguns outrosambientes.

O Game Project Manager (GPM) é um programa bem simples que automatiza a criação de documentação de jogos e permite a visualização e manipulação de documentos já criados.
